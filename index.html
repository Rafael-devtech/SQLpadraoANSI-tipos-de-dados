Dados Numéricos

INTEGER

Definição

Pode ser usado como "INT" ou "INTEGER", armazena números inteiros sem ter um tamanho fixo, mas geralmente ocupa em torno de 4 bytes.

Exemplos

  • Ex1: 1234567
  • Ex2: 5475543

SMALLINT

Definição

Número inteiro menor que o INT, que normalmente ocupa 2 bytes, mas também não tem tamanho fixo.

Exemplos

  • Ex1: 15
  • Ex2: 6787

BIGINT

Definição

Número inteiro maior que o INT, normalmente ocupa 8 bytes mas não tem tamanho fixo.

Exemplos

  • Ex1: 78.436.785.647
  • Ex2: 25.649.275.643.855

DECIMAL

Definição

Número com precisão fixa, é usado como DECIMAL(P,S) onde "p" determina a quantidade de caracteres totais e "s" a quantidade de caracteres nas casas decimais.

Exemplos

  • Ex1: DECIMAL(5,2) pode abrigar: 123,45
  • Ex2: DECIMAL(10,5) pode abrigar: 53752,74095

FLOAT

Definição

Número com casa decimal que pode variar na precisão, é usado como FLOAT(n), onde "n" define o número de bits para armazenar a parte fracionária, e pode variar de 1 a 53. A imprecisão na parte fracionária acontece pois é guardada como notação científica na máquina.

Exemplos

  • Ex1: FLOAT(4) pode guardar: 3,12
  • Ex2: FLOAT(12) pode guardar: 9,54631

REAL

Definição

Número real, que pode ter variação na precisão, e tem uma precisão menor que o DOUBLE PRECISION.

Exemplos

  • Ex1: 423,457
  • Ex2: 857,846

DOUBLE PRECISION

Definição

Número com casa decimal, pode ter variações na precisão e tem uma precisão simples e dupla (8 bytes para precisão), é ideia para aplicações científicas e estatísticas.

Exemplos

  • Ex1: 424,235
  • Ex2: 876,9538

Dados de Caractere

CHAR

Definição

É usado como CHAR(n), sendo "n" o tamanho fixo (que pode variar de 0 a 225 caracteres). CHAR pode armazenar qualquer string.

Exemplos

  • Ex1: CHAR(3) pode armazenar: 1a3
  • Ex2: CHAR(10) pode armazenar: 7m9463of75

VARCHAR

Definição

É usado como VARCHAR(n), sendo "n" o tamanho máximo, dentro desse espaço o tamanho pode variar (até 65535 caracteres). VARCHAR pode armazenar qualquer string .

Exemplos

  • Ex1: VARCHAR(3) pode armazenar até: 7c8
  • Ex2: VARCHAR(7) pode armazenar até: 64frg9j

Dados Binários

BINARY

Definição

Guarda dados binários, ou seja cadeias de bytes, é usado como BINARY(n) onde "n" representa a quantidade exata de bytes de espaço para armazenamento.

Exemplos

  • Ex1: Vídeos
  • Ex2: Imagens

VARBINARY

Definição

Guarda dados binários, ou seja cadeias de bytes, é usado como VARBINARY(n) onde "n" representa a quantidade máxima de bytes de espaço para armazenamento.

Exemplos

  • Ex1: Imagens

DATE

Definição

Armazena datas (dia, mês e ano), estes são separados por hífen (-), barra (/) ou ponto (.). A sequência entre dia, mês e ano pode variar de ordem.

Exemplos

  • Ex2: 10-12-2018
  • Ex3: 30.06.2020
